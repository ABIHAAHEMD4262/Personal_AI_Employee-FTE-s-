# Create Plan Skill

Generate structured Plan.md files with checkboxes for multi-step tasks.

## Usage

### Via Qwen Code

```bash
# Generate plans for all pending items
qwen "Create plans for all items in AI_Employee_Vault/Needs_Action"

# Generate plan for specific file
qwen "Create a plan for FILE_invoice.pdf in Needs_Action"
```

### Via Python Script

```bash
# Generate plans for all pending items
python skills/actions/create-plan/create_plan.py AI_Employee_Vault

# Verbose output
python skills/actions/create-plan/create_plan.py AI_Employee_Vault --verbose

# Generate plan for specific file
python skills/actions/create-plan/create_plan.py AI_Employee_Vault --file FILE_invoice.pdf
```

## Workflow

1. **Scan** `/Needs_Action` folder for action files
2. **Read** metadata and content from each file
3. **Analyze** task type and complexity
4. **Generate** Plan.md with:
   - Task objective
   - Step-by-step checklist
   - Required resources
   - Estimated time
5. **Save** to `/Plans/` folder
6. **Link** plan back to original action file

## Plan Template

```markdown
---
type: action_plan
created: 2026-02-27T10:30:00Z
status: pending
priority: normal
source: FILE_invoice.pdf
estimated_time: 15 minutes
---

# Plan: [Task Name]

## Objective
Clear statement of what needs to be accomplished.

## Steps
- [ ] Step 1: Description
- [ ] Step 2: Description
- [ ] Step 3: Description

## Required Resources
- Links to relevant files
- External references
- Contact information

## Notes
Additional context or considerations.

---
*Generated by AI Employee Create Plan Skill*
```

## Task Type Templates

| Type | Steps Generated |
|------|-----------------|
| email_reply | Read, Draft, Review, Send |
| invoice | Identify client, Calculate amount, Generate PDF, Send |
| meeting_notes | Extract action items, Assign owners, Schedule follow-ups |
| document_review | Read, Summarize, Extract key points, File |
| payment | Verify amount, Check approval, Process, Confirm |

## Output Location

Plans are saved to:
```
/Vault/Plans/
├── PLAN_invoice_client_a_2026-02-27.md
├── PLAN_email_reply_2026-02-27.md
└── PLAN_meeting_notes_2026-02-27.md
```

## Integration

Works with:
- `process-drop` - Triggers plan creation for complex files
- `approval-workflow` - Plans can include approval steps
- `schedule-task` - Plans can be scheduled for execution
